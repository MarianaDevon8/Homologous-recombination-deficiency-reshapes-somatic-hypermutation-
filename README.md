# Homologous-recombination-deficiency-reshapes-somatic-hypermutation-
Data availability-Homologous recombination deficiency reshapes somatic hypermutation patterns and reveals mutational signatures in antibody repertoires

This repository contains the code, processed data, and reproducible workflows used in the study:

“Homologous recombination deficiency reshapes somatic hypermutation in the human antibody repertoire.”
The project investigates how germline pathogenic variants in BRCA1/BRCA2 (homologous recombination deficiency, HRD) influence:

-Antibody repertoire structure
-Clonal diversity
-Somatic hypermutation (SHM)
-Mutational signatures (COSMIC-style)
